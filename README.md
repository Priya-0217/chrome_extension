# My Chrome Extension

This is a simple Chrome extension that adds a custom icon to the Chrome toolbar and opens a popup when clicked.

## Features

* Custom icon in the Chrome toolbar
* Popup page that opens when you click the icon

## Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/YOUR_USERNAME/my-chrome-extension.git
   ```

2. Open Google Chrome and navigate to:

   ```
   chrome://extensions/
   ```

3. Enable **Developer mode** using the toggle in the top-right corner.

4. Click **Load unpacked** and select the extension folder you just cloned.
